# iChat = Interactive Chat

## 어떤 기능을 넣을까??

- 실시간 투표 기능 - 제스처를 이용한 투표
- 방장의 실시간 질문 선택 및 답변 기능
- 다이나믹 아일랜드

## Structure

- Cloud: AWS EC2, NGINX
- DB: Superbase(postgresql), Redis
- Client: React(Vite)
- Server: Nest
